📁 File Handling Utility in Java
This project is a simple File Handling Utility developed using Java as part of an internship task. The main purpose of this project is to understand how file operations work in Java and to apply basic programming concepts in a practical way.
The application is a menu-driven console program that allows users to perform different operations on a text file. It is designed to be easy to use and helps beginners learn how to handle files step by step.

🎯 Objective
> The objective of this project is to create a program that can:
> Write data into a file
> Read data from a file
> Append new data to an existing file
> Modify the file content by replacing words

⚙️ Features
1. Write File
This option allows the user to enter some text and save it into a file. If the file already has content, it will be overwritten with the new data.
2. Read File
This feature reads the content of the file and displays it on the screen. It helps the user check what is stored inside the file.
3. Append File
Using this option, the user can add new content to the file without deleting the existing data. The new content is added at the end of the file.
4. Modify File
This feature allows the user to replace a specific word with another word in the file. The program reads the file, makes the changes, and updates the file.
5. Menu-Driven Program
The program runs in a loop and shows a menu with different options. The user can choose any operation by entering the corresponding number.
